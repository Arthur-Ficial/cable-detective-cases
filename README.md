# Cable Detective - Cases

Public bug reports and feedback for **Cable Detective for macOS**.

## How to file a case

The easiest way is the in-app button **"Did we get something not quite right?"** on any report. It copies a redacted version of the report to your clipboard and opens the feedback form at <https://cable-detective.franzai.com/feedback/>. You review what gets sent, attach photos, and submit. The form files an issue here.

You can also file an issue directly using the template below. **Please do not paste raw IOKit dumps that contain your Mac's serial number, hardware UUIDs, or hostname** - the in-app button strips these for you automatically.

## What goes in a case

- **What did the app get wrong?** (e.g. "said USB-C 2.0, but this is a Thunderbolt 4 cable")
- **What did you expect?** (e.g. "40 Gbps, USB4 logo, marked TB4 on the connector")
- **Photos** of the cable, connectors, and any markings - very helpful
- **Redacted scan report** copied from the app

We triage cases roughly weekly. If you provided a GitHub @handle, you'll get a notification on updates.
